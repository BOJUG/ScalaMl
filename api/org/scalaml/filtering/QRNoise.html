org.scalaml.filtering

QRNoise

case class QRNoise(qr: XY, white: (Double) ⇒ Double) extends Product with Serializable

Case class that defines the white (Gaussian) noise q for the process and the white noise r for the measurement or observation devices. An instance of this class has to be provided implicitly in the scope of the Kalman filter instance (DKalman).

qr

tuples that define the mean values of the process and measurement noise

white

scalar function that specify the white noise distribution

Since

February 12, 2014

Exceptions thrown
IllegalArgumentException

if the white noise generator is undefined
