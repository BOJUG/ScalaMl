  2. class Likelihood[T] extends AnyRef

    Class that represents a likelihood for each feature for Naive Bayes classifier.
    The prior consists of a label (index), the mean of the prior of each dimension of the model, the standard deviation of the prior of each dimension of the model and the class likeliHood.
    The Naive Bayes assume that the dimension of the model are independent, making the log of the prior additive.

  4. final class NaiveBayes[T] extends PipeOperator[XTSeries[Array[T]], Array[Int]] with Supervised[T]

    Generic Binomial Naive Bayes classification class.

    Generic Binomial Naive Bayes classification class. The class is used for both training and run-time classification. The training of the model is executed during the instantiation of the class to avoid having an uninitialized model. A conversion from a parameterized array, Array[T] to an array of double, DblVector, has to be implicitly defined for training the model.
    As a classifier, the method implement the generic data transformation PipeOperator and the Supervised interface.
