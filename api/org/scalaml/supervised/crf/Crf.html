org.scalaml.supervised.crf

Crf

final class Crf extends PipeOperator[String, Double] with Supervised[String]

Generic class for the linear chained CRF for tagging words, N-Grams or regular expression. The class define a Feature generator class that inherits the default implementation FeatureGenImpl of iitb features generator. The class assumes the training sequences are loaded from file with *.raw and *.tagged extensions.

Since

April 3, 2014
